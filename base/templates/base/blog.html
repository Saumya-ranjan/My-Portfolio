Django Overview Django
Django
  • Django Overview
  • Date : 05 December 2021
  • Categories : Django [Python]

Django is a Python web framework to build and design web applications.
Django is based on MVT architecture that is model,view and Template.

Models :- Models are basically the interface to handle data. its represented by a database where you can pass and get data from.
Whenever we create a Model, Delete a Model, or update anything in any of models.py of our project. We need to run two commands makemigrations and migrate. makemigrations basically generates the SQL commands for preinstalled apps (which can be viewed in installed apps in settings.py) and your newly created app’s model which you add in installed apps whereas migrate executes those SQL commands in the database file. So when we run, Python manage.py makemigrations SQL Query to create above Model as a Table is created and Python manage.py migrate creates the table in the database.

Views :- The View is the user interface — what you see in your browser when you render a website. It is represented by HTML/CSS/Javascript and Jinja files.

Template :- A template consists of static parts of the desired HTML output as well as some special syntax describing how dynamic content will be inserted.

Virtual Environment Python
virtual env for python
  • Virtual env
  • Date : 05 December 2021
  • Categories : Python

Virtualenv is the backbone of python environment. It holds all different libraries related to each project so the user don't have to worry about different versions clashing and putting the project into an error state.
When u start a project it is required to create a virtualenv so that u can work on that project solely. Installing its dependencies only rather than thinking about global environment.

To run virtual environment in your project you have to follow this steps:

1. pip install virtualenv
2. virtualenv --version
3. virtualenv (name of virtual env)
4. source (name of virtual env)/scripts/activate
5. deactivate --to deactivate virtual env
